# MVP 开发清单

## 第一阶段

- 搭建后端工程骨架
- 提供前端联调用 Mock API
- 搭建 Flutter 入口工程
- 固化学生端、家长端、老师端页面信息结构

## 第二阶段

- 账号与角色体系
- 监护人同意与学生知情同意
- 邀请码绑定
- 学校与班级导入

## 第三阶段

- 每日自评
- 筛查模板与答题提交
- 风险分级 Mock 到真实规则
- 训练计划生成与完成状态更新

## 第四阶段

- AI 陪练后端安全网关
- 表情/行为状态观察建议
- 消息提醒
- 家长摘要接口与页面联动
- 老师预警状态更新与复核入口

## 第五阶段

- MySQL 8 接入
- Redis 接入
- Flyway 迁移
- 审计日志
- 删除与导出机制

## 当前建议优先实现

1. 学生登录与绑定
2. 同意与授权
3. 筛查模板与提交流程
4. 今日训练计划
5. 老师预警单 Mock
